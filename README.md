## Функционал: 
 
* Функция лайка
* Добавления карточки
* Удаления карточки с подтверждением
* Попап просмотра карточки
* Попап редактирования профиля
* Валидация форм редактирования профиля и добавления карточки

## Используемые технологии: 

* Nested БЭМ
* Git 
* HTML/CSS
* JS (ES6)
* WebPack
